JavaScript Operators - Operator are characters used to operate on variables

Arithmetic operators:

  • + Adding numbers and merging strings
  • - Subtraction
  • * Multiplying
  • / Dividing

  • % The remainder of the division (mod operator)

  • = Assigning value
  • += ie: x += y => meaning: x = x + y
  • -= ie: x -= y => meaning: x = x - y
  • /= as above
  • *= as above
  • %= as above
  • ++ Incrementation (auto adding +1 to each new item. fx: if you create a table with different cars models, you can have a column with a number starting from 1, and with each now row it automatically adds +1, so the second row holds 2, third row gets the number 3 etc.
  • -- Decrementation : the other way

there is actually 4 different types of the incrementation and decrementation

PRE means before, and POST means after. Using POST in an alert means, that the viewer see the value of x, but after they alertbox is closed, the javascript add +1 to it now.

  • x++ POST incrementation
  • x-- POST decrementation
  • ++x PRE incrementation
  • --x PRE decrementation
The modules % operator can be tricky to understand - use this link and go test and try changing the values on w3school

Relational operators (comparison operators)

They can be used fx when you want to check if someone who is typing in their password, and then need to type it again to make sure the did it correctly.

  • == Equals to/comparing
  • === Identical, but variables must be the same type too (intergers, strings, ...)
  • != Not equal too
  • < Less then
  • > Greater then
  • <= Less or equal
  • >= Greater or equal

Logical operators

  • 0 - False
  • 1 - True
  • ! - NOT
    • !(0) - change the result to 1
    • !(1) - change the result to 0
  • && Conjuction - AND
  • || Disjunction - OR (alternative)
