JavaScript Arrays

Arrays:

Arrays - series of values that are next to each other.
Arrays can be accessed easily. The first item in an arrays sits in the "box" that has the number 0 ( or [0] ). As with other programming languages, they start numbering from 0, instead of 1 as we usually do.

Useful things to know about arrays and how to work with them

  • Join() - is used to join elements by send argument
  • concat() - concatenate is connecting
  • length() - Super used. Counting/returning the length of an array so we can target it. fx by using -1 or +1. See javascript file for examples
  • push() - adding an element to the end of array
  • pop() - popping (removing) the last element
  • shift() - removing the first element
  • unshift() - moving all elements in the array by 1 and adding a new element at the beginning

  • sort() - sorting ascending/alphaberically (a to z). If you need to sort numbers we do a little more. ie: = numbers.sort(function(a ,b){return a - b;})
  • reverse() - is the opposite. Descending order (z to a)

  • slice() - it is slicing elements(in my script example) from <1,3] "<" means without the 1, and "]" means 3
  • splice() - argument1 : from which place we want to to start removing elements, Argument2 : how many elements we want to remove, [optional arguments] : elements you want to add after the position in argument1
